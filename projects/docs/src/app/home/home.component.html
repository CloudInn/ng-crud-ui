Ng Crud UI makes its possible to build reusable forms, form fields, formsets and views across your application.

Build Reusable Views

Ng Crud UI provides default component called a screen-wrapper which can build components passed to it and render them.

We provide two default components that you can customize and use to build your screens, first one is:

ListViewer: Build listing views for your models with just metadata

FormViewer: Build Creation/Updating forms for your models using your metdata definitions
